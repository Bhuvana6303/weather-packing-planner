# 🌤️ Weather Packing Planner

A responsive web application that provides real-time weather updates and smart packing recommendations based on the selected city's weather conditions. The application uses the OpenWeatherMap API to display current weather, forecasts, and location-based weather information with a clean, user-friendly interface.

---

## 🚀 Features

- 🌍 Search weather by city name
- 📍 Detect current location using Geolocation API
- 🌡️ Real-time weather information
- 📅 Multi-day weather forecast
- 🎒 Smart packing suggestions based on weather
- 🌙 Dynamic UI theme according to weather conditions
- 🔄 Temperature unit conversion (°C / °F)
- 🕘 Search history using Local Storage
- 📱 Fully responsive design for desktop and mobile

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/Bhuvana6303/weather-packing-planner.git
```

2. Navigate to the project folder

```bash
cd weather-packing-planner
```

3. Open `index.html` in your browser.

No additional installation is required.

---

## 🔑 API Configuration

This project uses the **OpenWeatherMap API**.

1. Create an account at https://openweathermap.org/

2. Generate an API Key.

3. Replace the API key inside `app.js`.

```javascript
const API_KEY = "YOUR_API_KEY";
```

## 📖 How It Works

1. Enter a city name or use your current location.
2. The application retrieves weather information from the OpenWeatherMap API.
3. Current weather and forecast are displayed.
4. Packing recommendations are generated based on:
   - Temperature
   - Weather conditions
   - Wind speed
   - Humidity
5. Previous searches are stored locally for quick access.
